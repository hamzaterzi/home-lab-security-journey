# home-lab-security-journey

This repository documents my hands-on security practice across two main areas:

## 1. Local Home Lab
A VirtualBox-based lab built with:
- Ubuntu Server
- Kali Linux
- Windows Server

Topics covered:
- SSH hardening and key-based authentication
- UFW and Windows Firewall
- Nginx deployment
- Fail2ban protection
- Log analysis on Linux and Windows
- Nmap-based service validation
- Bash automation and cron scheduling
- Python health checks, JSON reporting, parsing, warning logic, and report comparison

## 2. AWS Security Lab
AWS-focused security practice covering:
- EC2 secure setup
- IAM fundamentals
- S3 security
- Security Groups and network exposure
- Logging and monitoring
- Least privilege and hardening concepts

## Repository Structure
- `local-homelab/` → local virtualization-based lab notes and scripts
- `aws-security-lab/` → AWS-focused security notes and labs

## Goal
To build practical foundations in Linux, Windows administration, scripting, monitoring, networking, and cloud security.
