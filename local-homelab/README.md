# Local Home Lab

This section documents my local security and administration lab built in VirtualBox using Ubuntu Server, Kali Linux, and Windows Server.

## Scope
- Linux administration
- Windows Server basics
- SSH hardening
- UFW and Windows Firewall
- Nginx deployment
- Fail2ban protection
- Log analysis
- Nmap validation
- Bash automation
- Cron-based reporting
- Python health checks
- JSON reporting, parsing, warning logic, and comparison

## Lab Topology
- Ubuntu Server
- Kali Linux
- Windows Server
- Host-only internal network
- NAT for internet access where needed

## Key Outcomes
- Built and hardened an Ubuntu Server VM
- Integrated a Windows Server VM into the same internal network
- Validated exposed services with Nmap
- Built Bash and Python-based reporting workflows
- Automated recurring reports with cron
